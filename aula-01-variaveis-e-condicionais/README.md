# Aula 01 - Variáveis e Condicionais

- Variáveis
- Strings

- echo
- if

## Exercícios

### Cadastro simples

Exibe:

- Nome
- Idade
- Cidade
- Profissão
- Objetivo profissional

### Validação de nota

Regras:

- Nota >= 7 → Aprovado
- Nota >= 5 e < 7 → Recuperação
- Nota < 5 → Reprovado
